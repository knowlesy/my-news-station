# Security Policy

If you find a vulnerability, please report it privately via
[GitHub Security Advisories](https://github.com/knowlesy/my-news-station/security/advisories/new)
rather than opening a public issue. Reports are looked at on a best-effort basis —
this is a self-hosted hobby project, but security reports are taken seriously.

Note that the station is designed to run on a **trusted home network**: the dashboard,
API, and OPDS catalog have no authentication. Do not expose them directly to the
internet without putting an authenticating reverse proxy in front.
