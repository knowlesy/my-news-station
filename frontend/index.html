CROSSPOINT X4 DEVICES
ⓘ Requires CrossPoint firmware with the device joined to this network. Stock-firmware X4s only listen on their own WiFi hotspot, which the server cannot reach — their light will stay 🔴.
OPDS CATALOG (E-READER PULL)
Catalog URL: …
Set up on the X4 (CrossPoint firmware):
  1. Join the X4 to this network (same WiFi as the station)
  2. On the device: Library → OPDS catalogs → Add catalog
  3. Enter the catalog URL above — leave username/password empty (no auth)
  4. Open the catalog: newest editions are listed first; pick one to download it straight to the device
Also works in KOReader, Calibre, Moon+ Reader, or any OPDS 1.2 client. Once pulling is proven, the push-based "Send to X4" above can be retired.
